# PTTK_07_S2_24_25

## Demo video Youtube [ở đây](https://youtu.be/qF_j6w9QolI)

Sử dụng bằng cách clone toàn bộ repo về và mở trong Visual Studio hoặc phần mềm tương tự. 
Chạy các file sql trong MS SQL Server theo thứ tự scriptACCI -> insertNewACCI -> insertNew2ACCI -> các file trigAndProc.
Sau đó vào Visual Studio, mở solution, vào file Program.cs và chọn Build/Start để chạy.

App PTTK_07 được lập trình bằng C\# Winform, back end là Microsoft SQL Server. Khi chạy file thì người dùng nên install package Microsoft.Data.SqlClient trước khi sử dụng.
